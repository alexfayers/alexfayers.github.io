Things I've made

Here's some of the cool things that I've created

Simply Budget

Simply Budget is an Android app I made in order to learn React Native and solve issues that I had with other budgeting apps.

Raspberry Pi Smart Light

I couldn't afford real smart lights for my room due to being a student, so I created my own out of things I already had: a Raspberry Pi and a USB powered light.

Toilet Occupancy Detector

I went to use the toilet and found it occupied too many times, so using a door closure sensor and some Python code I made a website that lets me know if the toilet is in use. The site also uses previous data collected to predict when the toilet will be free if it is in use.

ComputerCraft Quarrying System

I installed the ComputerCraft Minecraft mod and consequently spend more time developing an automated quarrying system than actually playing the game that I was developing the system for.
